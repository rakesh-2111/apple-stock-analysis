## Project Overview
This project analyzes Apple Inc. (AAPL) stock prices using historical market data.
The goal is to understand price trends, volume patterns, and moving averages.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

## Key Insights
- Long-term upward trend in Apple stock
- High volatility during market uncertainty
- Moving averages help identify trend direction

## Dataset
Historical stock price data for Apple (AAPL)

## Conclusion
Apple stock shows consistent growth with periodic corrections, making it a strong long-term investment candidate.
